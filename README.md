# WanderLust

A small Express.js project for listing places and reviews.

## Features

- Listings CRUD
- User authentication
- Reviews for listings

## Prerequisites

- Node.js 14+ (or newer)
- npm

## Setup

1. Install dependencies:

```bash
npm install
```

2. Run the app:

```bash
npm start
```

## Project files

- `app.js` — application entry
- `routes/` — route handlers
- `models/` — Mongoose models
- `views/` — EJS templates
