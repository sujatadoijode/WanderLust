# WanderLust

A web application built with HTML, CSS, JavaScript, Node.js and Express.js.

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/sujatadoijode/WanderLust.git

## Features

- Listings CRUD
- User authentication
- Reviews for listings
- MCV(models, controllers, views)
- MAP API (For Navigation)

## Prerequisites

- Node.js 14+ (or newer)
- npm

## Setup

1. Install dependencies:

```bash
npm install
```

2. Run the app:

```bash
npm start
```
The site will be available at http://localhost:8080
## Project files

- `app.js` — application entry
- `routes/` — route handlers
- `models/` — Mongoose models
- `views/` — EJS templates
- `controllers/`-- controller functions for handling business logic
- `public/` — static assets (CSS, JS, images)
